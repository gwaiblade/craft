# craft

Personal reference library — reusable skills and workflow protocols, written for my own use and for AI sessions to fetch directly.

## Structure
```

skills/       Single capabilities — reference docs read before or during work
protocols/    Multi-step workflows — runnable procedures with defined deliverables

```

One folder per entry: `skills/[name]/SKILL.md`, `protocols/[name]/PROTOCOL.md`.

## Usage

Everything here is fetchable at raw URLs, e.g.:
```

https://raw.githubusercontent.com/gwaiblade/craft/main/skills/remove-background/SKILL.md

```

Drop a raw URL into an AI session to load a skill or protocol as working context.

## Notes

Formats and inclusion rules are governed by private canons (CANON_SKILL, CANON_PROTOCOL). This repo holds craft material only — nothing here depends on private context.
