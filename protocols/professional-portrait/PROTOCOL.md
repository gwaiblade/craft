# Professional Portrait

**Purpose:** Transform an existing photograph into a professional portrait while preserving the person's identity with maximum accuracy. The result should look captured by an experienced lifestyle portrait photographer — not generated by AI.

**When to run:** A photo needs to become a professional portrait — LinkedIn, company websites, speaker profiles, press kits, personal branding.

**Inputs:** Collected interactively — see the steps. Subject photograph(s), background image, framing, pose, clothing, mood, photography style.

**Output:** One portrait image, reviewed and confirmed before generation, delivered side-by-side with the reference.

**Companion:** `VERIFICATION.md` (same folder) — identity verification tooling and agentic execution spec. Reference material only. Never paste it into a run.

**Skills called:** none

**Version:** 1.6

# CORE PRINCIPLE

Identity is sacred.

Improve the photograph, not the person.

Never redesign the person.

The portrait should immediately be recognizable by family, friends and colleagues.

Priority Order:

1. Facial Identity
2. Facial Expression
3. Hairstyle
4. Glasses & Accessories
5. Body Proportions
6. Pose
7. Clothing
8. Lighting
9. Background Integration

Never change this priority.

---

# INTERACTIVE MODE

This protocol is intentionally conversational.

Do NOT ask for every input at once.

Only ask ONE question at a time.

Wait for the user's reply before proceeding.

Never skip a step unless the user explicitly requests it.

---

# STEP 1 — SUBJECT PHOTO

Ask:

> Please upload the primary photograph of the person you would like to turn into a professional portrait.

This photograph will determine:

- Facial identity
- Hair
- Facial expression
- Glasses
- Body proportions

Do not ask any other questions yet.

Wait until the image has been uploaded.

**On receipt, check the photograph quality:**

- The face occupies at least 30–40% of the image height
- The face is sharp and in focus — especially the eyes
- Lighting on the face is reasonably even
- The face is not heavily obscured
- No portrait-mode artifacts around the face or hair
- Normal or telephoto perspective preferred over wide-angle selfies

A sharp, well-lit reference produces a portrait that truly looks like the person. If the photo falls short, say what would help and ask for a better one — a stronger reference now is worth more than any adjustment later.

Only after receiving a good photograph proceed to Step 2.

---

# STEP 2 — ADDITIONAL FACE REFERENCES

Ask:

> Would you like maximum facial accuracy?

Options:

**A — Yes (Recommended — Default)**

Upload 2–5 additional photographs.

Recommended:

- Front view
- Left 45°
- Right 45°
- Smiling
- Neutral

Explain:

These photos are used only to preserve identity.

Not composition.

If any detail differs between photos — glasses, hairstyle, facial hair — the primary photograph decides.

---

**B — No**

Continue using only the primary photograph.

If the user declines, explain:

> A single image can still produce a good portrait, but multiple references generally improve identity preservation because they provide additional facial geometry from different viewpoints.

Wait for completion before proceeding.

---

# STEP 4 — PORTRAIT FRAMING

Ask:

> Which framing would you like?

Options:

1. Headshot

2. Chest Up

3. Waist Up (Recommended)

4. Three Quarter

5. Full Body

Note when offering: tighter framings render the face with the most detail and the truest likeness. Full Body renders the face small — choose it when the full figure matters more than maximum facial fidelity.

Wait for selection.

---

# STEP 5 — POSE

Ask:

> What would you like to do with the pose?

Options:

### A — Preserve

Maintain the original pose exactly.

Only improve posture naturally.

---

### B — Refine

Keep the same pose.

Improve it slightly.

Examples:

- Better posture
- Relaxed shoulders
- More balanced stance
- More natural weight distribution

---

### C — New Pose

Create a completely new pose.

If selected, ask:

> Please describe the type of pose you would like.

Examples:

- Hands in pockets
- One hand pocket
- Arms relaxed
- Folded arms
- Sitting casually
- Leaning
- Walking naturally
- Creative founder
- Editorial lifestyle
- Relaxed leadership

Note when C is chosen: a new pose keeps the expression's character — the same emotion, the same smile — carried naturally into the new position, rather than an exact copy of the original geometry.

Wait for the answer.

# GENERATION RULES

Once confirmed, apply the following rules.

**Image anchor:** When generating, use the uploaded photographs from Steps 1–2 as the direct face reference. Never substitute a text description of the face for the images themselves — the conversation describes the styling; the photographs define the person.

---

# PREFERRED EDITING STRATEGY

Identity-preserving editing beats face regeneration.

Whenever the image model supports editing, treat the uploaded face as locked.

Only modify:

- Background
- Lighting
- Exposure
- Colour grading
- Crop
- Depth of field
- Clothing (if requested)
- Minor posture refinement

Avoid regenerating facial structure.

Regenerate the face only when the requested change makes editing impossible — and even then, anchor every facial detail to the reference photographs.

---

# IDENTITY LOCK

Maximum Identity Preservation.

The face must remain as close as possible to the uploaded photographs.

Preserve:

- Face shape
- Facial asymmetry
- Hairline
- Hair direction
- Hairstyle
- Forehead
- Eyes
- Eyelids
- Eyebrows
- Nose
- Lips
- Smile
- Teeth visibility
- Chin
- Jawline
- Ears
- Skin texture
- Wrinkles
- Freckles
- Moles
- Facial hair
- Glasses
- Age
- Weight
- Ethnicity

Do NOT:

- Beautify
- Stylize
- Sharpen the jaw
- Slim the face
- Change eye size
- Smooth the skin excessively
- Symmetrize the face
- Change ethnicity
- Make the subject younger
- Make the subject older

The objective is:

"A better photograph of the exact same person."

Never:

"A better-looking version."

---

# EXPRESSION LOCK

Maintain the original expression unless the user explicitly requests otherwise.

Preserve:

- Smile intensity
- Eye openness
- Eye direction
- Lip position
- Eyebrow position
- Emotion

Do NOT:

- Increase the smile
- Add confidence
- Add seriousness
- Change facial emotion

The portrait should feel like the exact same moment photographed more professionally.

(With a new pose from Step 5C: carry the expression's character — same emotion, same smile intensity — naturally into the new head position.)

# OUTPUT PLACEMENT

If the destination is known, compose for it:

- **Circle-crop avatar** (LinkedIn, Slack, GitHub): face centred, generous headroom, nothing critical in the corners — the face must survive a centre circle crop.
- **Landscape** (banners, headers): subject off-centre is acceptable.
- **General professional** (speaker profiles, about pages): native aspect ratio, no forced crop.

If unknown, compose circle-safe — it costs nothing and survives every placement.

# IF THE PORTRAIT NEEDS ANOTHER ATTEMPT

A stronger portrait comes from simplifying, not from repeating the same request.

**If the likeness is off:**

1. Set Clothing back to Preserve and try again.
2. Still off — set Pose back to Preserve as well.
3. Still off — a sharper, closer, better-lit reference photograph is the answer. Ask for one.

**If the likeness is right but the expression is off:**

Restate the Expression Lock explicitly and regenerate with everything else unchanged.

**If hands or body look wrong:**

Simplify the pose — hands in pockets or out of frame — or tighten the framing one level.

**If the background changed:**

Restate the background as fixed input; adjust only perspective, colour temperature, focus, and light direction.

Three attempts maximum. If the third attempt still misses, say plainly which input is limiting the result — and explain that the limitation is the model's tendency to synthesize a new face rather than preserve exact geometry. Do not continue regenerating indefinitely.

Identity always wins. A portrait with a perfect background and the wrong face is a failed portrait. A portrait with a slightly imperfect background and the right face is a fixable one.

# SUCCESS CRITERIA

If someone who knows the subject personally sees the portrait, their first reaction should be:

> "That's a fantastic photo of you."

—not—

> "That doesn't quite look like you."

The protocol succeeds only when the person's identity has been preserved while the photography has been elevated.
